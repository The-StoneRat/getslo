# getslo.cam

Static landing page and privacy policy for the Slo Android app.

- `index.html` — landing (hero + how it works)
- `privacy.html` — privacy policy (served at `/privacy`)
- `styles.css` — shared styles

Hosted on GitHub Pages, custom domain `getslo.cam` (see `CNAME`).
No build step: edit the HTML/CSS and push to `main`.
